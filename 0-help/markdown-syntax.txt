## 2. Text Formatting

- **Bold**: `**bold**` → **bold**
- *Italic*: `*italic*` → *italic*
- ~~Strikethrough~~: `~~strikethrough~~` → ~~strikethrough~~
- `Inline Code`: `` `inline code` `` → `inline code`

## 3. Lists

### Unordered List
- Item 1
- Item 2

  - Sub-item 2

### Ordered List
1. First item
2. Second item
   1. Sub-item 1
   2. Sub-item 2
